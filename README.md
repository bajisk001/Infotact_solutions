# 🔐 Infotact Solutions – Cybersecurity Training & Projects

## 📌 Overview

This repository contains my complete learning journey and hands-on projects completed as part of the **Infotact Solutions Cyber Defense Operations Center (CDOC) training program**.

The program is designed to cover **end-to-end cybersecurity concepts**, including Red Team, Blue Team, and Security Operations.

---

## 📅 Training Roadmap

### 🔹 Week 1: Fundamentals & Linux for Security

* CIA Triad (Confidentiality, Integrity, Availability)
* Threat Actors & Attack Vectors
* OSI vs TCP/IP Models
* Linux Basics & Bash Scripting

🛠️ **Tasks:**

* Setup Virtual Lab (Kali Linux & Metasploitable)
* Create Rules of Engagement
* Develop Bash automation scripts

---

### 🔹 Week 2: Network Reconnaissance & Scanning

* OSINT (Google Dorking, WHOIS, DNS)
* Nmap Scanning Techniques
* Packet Analysis using Wireshark

🛠️ **Tasks:**

* Perform network discovery
* Identify open ports & services
* Analyze PCAP files

---

### 🔹 Week 3: Vulnerability Assessment

* Vulnerability Lifecycle & CVSS
* Enumeration (SMB, SNMP, LDAP)
* OpenVAS / Nessus Scanning

🛠️ **Tasks:**

* Run vulnerability scans
* Analyze reports
* Identify critical vulnerabilities

---

### 🔹 Week 4: System Hacking & Exploitation

* Metasploit Framework
* Password Attacks (Hydra, John)
* Privilege Escalation

🛠️ **Tasks:**

* Gain system access
* Perform exploitation
* Maintain access

### 🔹 Week 5: Web Application Security (OWASP)

* HTTP Protocol & Sessions
* SQL Injection (SQLi)

### 🔹 Week 6: Advanced Security Concepts

### 🔹 Week 7: SOC & Incident Response

### 🔹 Week 8: Capstone & Career Readiness

🛠️ **Tasks:**
